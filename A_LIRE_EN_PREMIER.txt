───────────────────────────────────────────────────────────────────────
  1. CE QU'EST LE PROJET
───────────────────────────────────────────────────────────────────────

"Awakened" : une PWA (Progressive Web App) de fitness inspirée de Solo
Leveling. Elle mêle suivi d'entraînement réel et mécaniques de jeu RPG
(Failles/rifts, monstres, boss, arbre de compétences, stats, rangs E→S,
compagnons, équipement avec raretés, narration).

Stack : JavaScript / HTML / CSS PUR (vanilla). AUCUN framework, aucun build
system. Le code tourne directement dans le navigateur.

L'utilisateur travaille SUR MOBILE (téléphone) et communique EN FRANÇAIS.
Réponds toujours en français.

  4. PROCÉDURE DE BUILD (à suivre à chaque modification)
───────────────────────────────────────────────────────────────────────

Base du projet : /home/claude/fitpro/   (ou là où le zip est extrait)
Sortie : un zip à présenter à l'utilisateur

Étapes :
  1. Faire la modification chirurgicale demandée
  2. Vérifier la syntaxe : node --check js/app.js  (et tout fichier .js modifié)
  3. Incrémenter le cache (exemple v154 → v155) :
       sed -i "s/awakened-v154/awakened-v155/g" sw.js
       sed -i 's|?v=154|?v=155|g' index.html
  4. Nettoyer : rm -f *.bak* js/*.bak*
  5. Zipper et copier vers le dossier outputs
  6. Présenter le zip à l'utilisateur

⚠️  TOUJOURS partir du zip LE PLUS RÉCENT fourni par l'utilisateur. Ne jamais
    repartir d'une ancienne version (sinon on perd tout le travail accumulé).
